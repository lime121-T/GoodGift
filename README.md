礼遇·GoodGift - 智能礼物推荐工具

项目简介：
一个轻量级的智能礼物推荐Web应用，帮助用户根据收礼人关系、预算、风格偏好等条件，快速获得合适的礼物建议。所有数据存储在浏览器本地，无需后端服务器。

主要功能：
- 智能推荐：根据收礼人关系、预算、风格、场合等条件推荐礼物
- 关系过滤：自动过滤不适合的礼物（如不会给妈妈推荐剃须刀）
- 换一批：可刷新推荐列表，自动排除已显示的礼物
- 细化指令：在原指令基础上追加新要求，重新推荐
- 礼物档案：记录送礼历史，支持反馈评价
- 重要日期：日历视图查看生日/纪念日
- 语音输入：支持语音描述需求
- 逛逛榜单：查看热门礼物排行榜

快速开始：
1. 下载 `index.html` 文件
2. 用浏览器打开即可使用
3. 所有数据保存在浏览器本地存储中

使用说明：

首页推荐
1. 在输入框描述收礼人信息（如“送给妈妈 预算200-500元”）
2. 可点击标签快速添加预算、风格、场合、收礼人
3. 点击「开始推荐」获取礼物建议

推荐结果页
- 送这个：记录选中的礼物
- 购买：演示链接（暂不跳转）
- 换一批：刷新推荐，避免重复
- 细化指令：补充要求后重新推荐

档案页
- 查看所有收礼人列表
- 点击日历图标查看重要日期
- 点击收礼人查看送礼历史
- 支持编辑反馈

技术栈：
- HTML5 + CSS3
- Tailwind CSS
- 原生JavaScript
- LocalStorage（数据持久化）

文件结构：
index.html           单文件应用，所有代码在一个文件中
